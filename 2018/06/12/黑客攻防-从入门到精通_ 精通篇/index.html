第十一章

黑客编程基础

  1. 常见的编程语言类型

    使用黑客语言进行编程时，需确定不同的语言类型在不同领域中的应用，下面将对不同的语言类型进行分组。

  • web page script languages：指网页代码，主要包括HTML、JavaScript、CSS、ASP、PHP和XML等。
  • Interpreted Languages(解释型语言)：主要包括Perl、python、REBOL和Ruby等，也常被称作script语言，通常被用于操作系统的沟通。这类语言的缺点是效率差、源代码泄露，不适合用来开发软件产品，一般用于网页服务器。
  • COMPLILING Languages(编译型语言)：C/C++、Java都是编译型语言，如要使用跨Linux和Windows平台的语言，则Delphi是最好的选择。
  • hybrid languages(混合型语言)：主要包括Java和C#，介于解释型和编译型之间，C#是.net最重要的语言，值得期待其后续发展。
  • assembly languages(汇编语言)：它是最接近于硬件的语言，但目前很少有人使用。

注意：

在学习编程时，需切记每种语言总是略学皮毛，又急匆匆的学习另一种语言，这样将一事无成，永远彷徨在初学者的门槛。

动态链接库–DLL

DLL是一个包含由多个程序同时使用的代码和数据的库，它不是可执行文件。DLL有助于共享数据和资源，多个应用程序可同时访问内存中单个DLL副本的内容，下面将对其进行介绍。

DLL的工作方式

动态链接库是一个包含模块集合的文件，任意数量的不同程序都可以使用它。将一个函数存储在一个DLL以后，就可以在不影响使用该函数的情况下修改它。只要到这个DLL中的这个函数的接口不变，那么程序就可以非常便捷的使用这个函数的新版本，而不需要重新编译或重新链接这些程序。但这也有一个缺点，容易将错误的DLL版本应用于程序。对于把DLL安装在Windows system文件夹中的应用程序来说，这是一个特别严重的问题。一些商业应用程序随意的将与程序相关联的DLL写入这个文件夹，而没有考虑具有相同名称的DLL被改写的可能性。这可能会干扰其他已经安装的应用程序，最坏的情况，会导致它们无法使用。

DLL的内容

  • DLL接口：不能访问DLL中包含的所有内容。外界只能看到明确标识为可从D的项。函数、类、全局静态变量和资源都可以从DLL导出，它们构成了D口。没有导出的所有内容都不能从外部访问。
  • DLLMain()函数： 尽管DLL不像独立程序那样是可执行的，但是它却包含一种特殊的main称作DLLmain()函数。第一次把DLL加载到内存中时，Windows将调用它，从而允许DLL在其内容被使用之前进行必要的初始化。Windows在从内存中DLL之前也要调用DLLmain（）函数，以支持DLL在必要时进行清理。

第十二章

网络威胁的清除和防御

间谍软件

间谍软件其实是一个“灰色区域”，所以并没有一个明确的定义。然而，正如同名字所暗示的一样，它通常被泛泛的定义为从电脑中搜集信息，并在未得到该用户许可的情况下，获取电子邮件地址和跟踪浏览习惯等。间谍软件之所以称为“灰色区域”，主要因为它是一个包罗万象的术语，包括许多与恶意程序相关的程序，而不是一个特定的类别。大多数的间谍软件定义不仅涉及广告软件、色情软件和风险软件程序，还包括许多木马程序，如backdoor Trojans、Trojan Proxies和PSW Trojans等。

提示：通常一些间谍程序随着共享软件安装到电脑里，监视电脑运行。用户的隐私数据和重要信息会被间谍软件捕获，并发送给黑客、商业公司等，这些间谍软件甚至能使用户的电脑被远程操控，组成庞大的“僵尸网络”。

流氓软件

定义：

流氓软件其实起源于国外的“badware”一词，“badware”的定义为：是一种跟踪用户上网行为并将用户的个人信息反馈给市场利益集团的软件。并且他们可以通过该软件向用户弹出广告。

特点

1. 强制安装
2. 难以卸载
3. 浏览器劫持
4. 广告弹出
5. 恶意收集用户信息
6. 恶意卸载
7. 恶意捆绑

流氓软件的分类

1. 木马程序
2. 恶意插件
3. 浏览器劫持
4. 恶意共享软件
5. 行为记录软件

第十三章

U盘攻击与防御

随着U盘的普及，其病毒也开始泛滥，用户在使用U盘时，经常会被U盘病毒进行攻击。U盘病毒又称autorun病毒，它是通过autorun.inf文件使用户所有的硬盘完全共享或中木马的病毒。

U盘病毒的工作原理

在U盘中写入病毒程序并更改autorun.inf文件，然后通过autorun.inf文件记录用户选择打开U盘的程序类型，如autorun.inf文件绑定了病毒程序，那么Windows就会运行这个程序从而引发病毒。另外，病毒也会主动监测插入电脑的U盘，并进行文件的关联，通过U盘病毒进行传播。

U盘病毒的隐藏方式

  • 伪装成其他文件
  • 作为系统文件隐藏
  • 其他新型的U盘病毒
  • 隐藏文件夹
  • 藏于系统文件中

U盘病毒的判断

电脑感染U盘病毒的症状

  • 出现EXE文件
  • 无法双击打开磁盘
  • 阻止程序运行
  • 无法启动任务管理器

U盘感染病毒的症状

  • 出现错误提示
  • 修改文件属性
  • 出现可疑文件
  • 存储空间减少
  • 不能停止设备
  • 出现可疑进程
